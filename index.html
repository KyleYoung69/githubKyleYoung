52 Card Deck:

Big fan of this Coffee Cup. I use it every day. Great for the good ol' wake and bake
The JavaScript Problem that I tackled was to make a 52 Card deck, be able to print it in order and be able to shuffle it and print the shuffled result. I changed what I wanted to do like 5 times. I was overambitious with what I wanted to do so I settled on this. It is a neat little program, showing off the math.random() function to shuffle. Adding buttons and results was relativly easy.
